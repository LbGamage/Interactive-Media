Responsive Design

Responsive Design allows people to access content across multiple device resolutions. The viewports should adjust logically on tablets, phones, and desktops of any resolution. Adopt a mobile-first approach, utilize scalable vector graphics (SVGs) for icons and illustrations, and implement three or more breakpoints for various screen sizes.
There are 3 main principles to follow:

  • Grid Systems: Uses relative sizing instead of fixed pixel dimensions.
  • Fluid Image: Images resize dynamically to fit their containers.
  • Media Queries:Adjust layouts based on screen size breakpoints.
